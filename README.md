# telegram_bots

## Install

  pip install -r requirements.txt
  
## Run locally 

To run default bot:

    python main.py
    
or explicitly 
    python main.py --bot ertaoz_bot
    
Check out help to run other bots

    python main.py -h
    python main.py --bot noshrevan_bot
    python main.py --bot qristefore_bot
    
    
## Run black formatter 

    black .

### Vscode -
